# 开局史莱姆

一个基于 Next.js + React + TypeScript 的回合制 RPG 小游戏

## 功能

- 🎮 **战斗页面**: 回合制战斗系统，人物 vs 怪物
- 🎒 **装备页面**: 装备栏 + 背包系统，支持分类筛选
- ✨ **技能页面**: 4个预制技能，可用金币升级

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

打开浏览器访问 http://localhost:3000

## 游戏玩法

1. 在战斗页面击败怪物获得经验和金币
2. 在装备页面管理装备，提升战斗属性
3. 在技能页面升级技能，增强战斗能力

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Zustand (状态管理)
