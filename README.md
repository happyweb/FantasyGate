# 幻境之门 Fantasy Gate

> 版本 v1.0.0

一款基于 Next.js + React + TypeScript 构建的单机回合制 RPG 手机网页游戏。

---

## 背景故事

千年前，光明神以神躯化环，才将暗影魔王与冥火军团封印在回廊深处。  
如今封印裂纹从雾原蔓延至王庭，五大战区相继失守。传说唯有重集五大传说套装，才能让封印纹阵再次闭合。

你在前线营地醒来，手边只有一把旧剑与一枚未熄的徽印，却被委以整片大陆最后的进攻权。  
从雾原前线到终焉王庭，每一场胜利都会点亮一道封印纹——走到尽头的人，才有资格直面焚冠冥皇。

---

## 游戏流程

### 1. 开场进入
- 启动游戏将显示背景叙事序列，带你进入世界观
- 输入或随机生成角色名，正式踏上征途

### 2. 五大战区循环
游戏分为 5 个难度递增的战区，每个战区包含 10 场战斗：

| 战区 | 名称 | 难度倍率 |
|------|------|---------|
| 第一区 | 比武草原（雾原前线） | ×1 |
| 第二区 | 裂岩荒境（裂石营地） | ×2 |
| 第三区 | 断垣古堡（古堡回廊） | ×3 |
| 第四区 | 风暴龙脊（高空裂谷） | ×4 |
| 第五区 | 深渊王座（终焉王庭） | ×5 |

通关第五区即击败最终 Boss，完成封印、赢得游戏。

### 3. 战斗系统（回合制）
- **普通攻击**：消耗无资源，每回合稳定输出
- **五大技能**：雷鸣斩、治愈之光、狂暴战吼、圣盾守护、封印术，各有 MP 消耗与独特效果
- 技能可在「技能」页用金币升级（最高 5 级），强化数值与效果
- 怪物会反击，并具有冻结、封印等状态效果
- 战斗过程有浮动伤害数字、击中震动、技能波纹等视觉反馈

### 4. 检查点与复活
- 每完成一个战区将自动存档为检查点
- 角色死亡后可选择从上一个检查点重新出发，保留进度

### 5. 装备与背包
- 5 个装备槽：头盔、武器、护甲、马匹、饰品
- 击败怪物随机掉落装备与消耗品
- 支持按分类筛选，装备/卸下即时生效，属性实时反映在战斗中

### 6. 商店
- 使用战斗获得的金币购买 HP / MP 补给道具
- 在两场战斗之间补充资源，延续征程

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

打开浏览器访问 http://localhost:3000（推荐移动端竖屏体验）

---

## 技术栈

- **Next.js 14** — App Router，SSR / CSR 混合渲染
- **React 18** — 组件化 UI
- **TypeScript** — 全量类型覆盖
- **Zustand** — 轻量全局状态管理，支持 localStorage 持久化
- **Tailwind CSS** — 原子化样式
- **Capacitor** — 可打包为 Android 原生 App
